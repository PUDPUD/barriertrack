Piechart Genereren

Hoe werkt het?

Om een piechart te genereren, voer je de details hieronder in. Dit omvat de begin- en eindtijd van de periode die je wilt analyseren, evenals het type event dat je wilt onderzoeken.

Gebruik de volgende formaten voor de invoer:

  • Begin Tijd: 2024-03-25 12:00:00
  • Eind Tijd: 2024-03-25 13:00:00
  • Event Type: Kies uit Accepted password, Disconnected, Failed password for invalid user, Failed password.
